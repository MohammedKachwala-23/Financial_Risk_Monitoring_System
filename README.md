# 📊 Financial Risk Monitoring System

A Machine Learning-based web application that analyzes financial transaction data to identify and classify risks into **High**, **Medium**, and **Low** categories. This system helps organizations monitor suspicious activities and gain insights through interactive visualizations.

---

## 🚀 Features

- 📂 Upload transaction dataset (CSV)
- 🤖 Automatic column detection
- ✍️ Manual column selection (if columns are not detected correctly)
- 📊 Risk classification (High / Medium / Low)
- 📈 Interactive dashboards and charts
- 🏢 Department-wise risk analysis
- ⚡ Fast and user-friendly interface

---

## 🧠 How It Works

1. User uploads a dataset containing transaction details  
2. System detects important columns (Amount, Department, etc.)  
3. Risk is calculated based on predefined rules / ML logic  
4. Transactions are classified into:
   - 🔴 High Risk  
   - 🟡 Medium Risk  
   - 🟢 Low Risk  
5. Results are displayed using charts and summaries  

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Data Processing:** Pandas, NumPy  
- **Machine Learning:** Scikit-learn  
- **Frontend:** HTML, CSS  
- **Visualization:** Chart.js / Matplotlib
